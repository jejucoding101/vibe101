# 🎯 화두 풀 — HTTP와 API

---

## 🟢 입문
1. **포켓몬 API 탐험** ⭐ — "포켓몬 이름 입력 → 이미지, 타입, 스탯 표시" (pokeapi.co)
2. **랜덤 고양이/강아지 사진** — "버튼 클릭 → 랜덤 동물 사진 표시"
3. **오늘의 명언 API** — "랜덤 영감의 명언을 가져와 카드로 표시"
4. **환율 조회** — "USD/EUR/JPY → KRW 환율 실시간 조회"
5. **국기 퀴즈** — "랜덤 국기 API → 어느 나라 국기인지 맞추기 게임"

## 🟡 도전
6. **GitHub 프로필 조회** ⭐ — "GitHub 사용자명 입력 → 프로필 카드 표시" (api.github.com)
7. **영화 검색** — "영화 제목 검색 → 포스터, 평점, 줄거리 표시" (OMDB API)
8. **뉴스 API** — "카테고리별 최신 뉴스 가져와 카드 목록으로 표시"
9. **지도 API** — "주소 입력 → 지도에 마커 표시" (Leaflet + Nominatim)
10. **공공데이터 API** — "버스 도착 정보, 미세먼지 등 공공 API 활용" (Python 추천)

## 🔴 심화
11. **API 응답 캐싱** — "같은 요청은 캐시에서 가져오는 최적화"
12. **여러 API 조합** — "날씨 + 뉴스 + 환율을 동시에 가져와 대시보드 구성"

## 💡 실생활
13. **출퇴근 날씨 알림** — "매일 아침 날씨 API 확인 → 우산 필요 여부 안내" (Python 추천)
14. **짤방 생성기** — "텍스트 입력 → 밈/짤방 이미지 API로 생성"
