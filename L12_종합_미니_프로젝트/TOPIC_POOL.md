# 🎯 화두 풀 — 종합 미니 프로젝트

> Part 2에서 배운 모든 기술을 조합하여 완성도 있는 미니 프로젝트를 만들어보세요!

---

## 🟢 입문 프로젝트

### 1. 단어장 암기 도구
> JSON으로 단어 관리 + 랜덤 퀴즈 + 정답률 통계 + CSV 내보내기
- 영어 단어, 일본어 단어, 프로그래밍 용어 등 자유롭게

### 2. 독서 기록 시스템
> 읽은 책(제목/저자/평점/감상) JSON 저장 + 장르별 통계 + 평점 차트

### 3. 운동 루틴 관리
> 요일별 운동 계획 JSON + 완료 체크 + 주간 달성률 차트

### 4. 식단/칼로리 트래커
> 음식별 칼로리 DB(JSON) + 일일 섭취 기록 + 주간 칼로리 차트

### 5. 할일 관리 도구 (콘솔)
> 할일 추가/완료/삭제 + JSON 저장 + 카테고리별 통계

---

## 🟡 도전 프로젝트

### 6. 레시피 관리 + 장보기 목록 ⭐ 추천
> 레시피 JSON DB + 레시피 선택 시 필요 재료 합산 → 장보기 목록 CSV 생성

### 7. 시간표/일정 생성기
> 요일×시간 그리드에 수업/일정 배치 + 충돌 검사 + 시각화

### 8. 퀴즈 출제 시스템
> 문제은행 JSON + 랜덤 출제 + 채점 + 오답 노트 + 정답률 차트

### 9. 간이 CRM (고객 관리)
> 고객 정보 JSON + 검색/필터 + 연락 이력 + 통계 리포트

### 10. 투표/설문 시스템
> 투표 항목 설정 + 투표 수집 + 실시간 결과 집계 + 차트 생성

---

## 🔴 심화 프로젝트

### 11. 일기장 + 감정 분석 (Python 추천)
> 일기 텍스트 저장 + 감정 키워드 분석(긍정/부정) + 월별 감정 추이 차트

### 12. 가계부 + 예산 관리
> 이번 강의 가계부 + 카테고리별 월 예산 설정 + 예산 대비 실 사용률 경고

### 13. 개인 위키 시스템
> 마크다운 문서 관리 + 태그 시스템 + 내부 링크 + 전문 검색
