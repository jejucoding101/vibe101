# 📌 12강: 종합 미니 프로젝트 — Part 2 총정리

> **핵심 포인트**: 6~11강에서 배운 기술(CSV, JSON, 파일 관리, 반복/조건, 텍스트 가공, 시각화)을 조합하여 **하나의 완성된 프로젝트**를 만드는 경험

---

## 📖 이론 (20분)

### 프로젝트 설계 프로세스

지금까지는 기능 하나씩 따로 만들었습니다. 이제 여러 기능을 **하나의 프로그램으로 조합**합니다.

```
1단계: 무엇을 만들까? (기능 목록 정리)
2단계: 데이터를 어떻게 저장할까? (CSV? JSON?)
3단계: 어떤 기능부터 만들까? (우선순위)
4단계: 하나씩 구현 + 테스트
5단계: 기능 연결 + 최종 테스트
```

### 이번 강에서 조합할 기술

| 강의 | 기술 | 프로젝트에서의 역할 |
|------|------|-------------------|
| 6강 | CSV | 데이터 내보내기/가져오기 |
| 7강 | JSON | 데이터 영구 저장 |
| 8강 | 파일 관리 | 자동 백업, 파일 생성 |
| 9강 | 반복/조건 | 데이터 처리 로직 |
| 10강 | 텍스트 가공 | 입력 검증, 검색 |
| 11강 | 시각화 | 통계 차트 생성 |

---

## 🔨 가이드 실습 (25분)

### 프로젝트: 개인 가계부 시스템

Antigravity에게 **단계별로** 요청합니다:

### Step 1: 데이터 구조 설계 (5분)

```
개인 가계부 앱의 데이터 구조를 JSON으로 설계해줘:
- 수입/지출 항목: 날짜, 카테고리, 금액, 메모
- 카테고리 목록: 식비, 교통, 쇼핑, 월급, 용돈 등
- 월별로 분류 가능한 구조
```

### Step 2: 핵심 기능 구현 (10분)

```
가계부의 핵심 기능을 만들어줘:
1. 항목 추가 (수입/지출, 카테고리, 금액, 메모)
2. 전체 내역 조회 (보기 좋은 표로 출력)
3. 기간별 조회 (특정 월의 내역만)
4. 카테고리별 합계
5. JSON 파일로 자동 저장/불러오기
JS와 Python 중 하나로 만들어줘.
```

### Step 3: 시각화 + 내보내기 (10분)

```
가계부에 추가 기능:
1. 카테고리별 지출 비율 → 파이 차트 생성
2. 월별 수입/지출 추이 → 선 그래프 생성
3. 전체 내역을 CSV로 내보내기
4. CSV에서 데이터 가져오기
```
