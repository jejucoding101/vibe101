데이터의 표준 언어 — key-value, 배열, 중첩 객체를 이해하고 활용하는 방법을 배웁니다.

이론

⏱ 20분

JSON이란 무엇인가요?

JSON은 JavaScript Object Notation의 약자로, 데이터를 주고받을 때 사용하는 세계 표준 형식입니다.

사람이 읽기 쉬운 텍스트 형태이면서, 동시에 컴퓨터가 자동으로 파싱할 수 있어서 웹 API, 설정 파일, 데이터 교환 등 거의 모든 곳에서 사용됩니다.

  profile.json

  "이름": "홍길동",        ← 문자열
  "나이": 25,              ← 숫자
  "취미": ["독서", "코딩", "등산"], ← 배열
  "주소": {               ← 중첩 객체
    "도시": "서울",
    "구": "강남구"

JSON을 작성할 때 반드시 지켜야 하는 5가지 규칙

JSON은 문법이 엄격합니다. 아래 규칙 하나라도 어기면 전체 파일이 에러가 되므로, 처음부터 정확하게 익혀 두는 것이 중요합니다.

1
키(key)는 반드시 "큰따옴표"로 감싸야 합니다.
2
값(value)에는 문자열, 숫자, 배열, 객체, true/false, null만 올 수 있습니다.
3
배열 [ ]은 여러 값을 순서대로 담는 목록입니다.
4
중첩이 가능합니다. 객체 안에 객체, 배열 안에 객체를 넣을 수 있습니다.
5
마지막 항목 뒤에 쉼표를 붙이면 안 됩니다. CSV와 다릅니다!

CSV vs JSON — 같은 데이터, 다른 표현 방식
