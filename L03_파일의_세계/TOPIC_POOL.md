# 🎯 화두 풀 — 파일의 세계

> 다양한 파일 형식을 만들고 읽고 변환하는 작업을 Antigravity로 해결해보세요.

---

## 🟢 입문 (바로 시도 가능)

### 1. 마크다운 자기소개 작성 ⭐ 추천
> "마크다운(.md) 형식으로 나의 자기소개 페이지를 만들어줘. 사진(placeholder), 이름, 취미, 목표 포함"

### 2. 일기장 TXT 생성
> "오늘 날짜를 파일명으로 하는 일기 템플릿 파일을 자동 생성하는 프로그램"

### 3. 설정 파일 만들기
> "게임 설정을 JSON으로 관리: 난이도, 소리 크기, 화면 밝기, 조작 키 설정"

### 4. 연락처 CSV 만들기
> "가족/친구 연락처를 이름, 전화번호, 이메일, 생일 형식의 CSV로 정리"

### 5. 읽기 쉬운 파일 목록
> "폴더 안의 모든 파일을 종류별(텍스트/이미지/문서 등)로 분류한 리포트 생성"

## 🟡 도전 (조금 더 생각 필요)

### 6. CSV ↔ JSON 변환기 ⭐ 추천
> "CSV 파일을 JSON으로, JSON 파일을 CSV로 변환하는 양방향 변환 도구"

### 7. 로그 파일 분석
> "서버 로그 파일(샘플)에서 ERROR가 포함된 줄만 추출하여 별도 파일로 저장"

### 8. 자막 파일(SRT) 시간 조정
> "SRT 자막 파일의 모든 타임스탬프를 2초씩 앞이나 뒤로 밀어주는 도구"

### 9. 단어 빈도수 분석
> "텍스트 파일을 읽어서 가장 많이 등장한 단어 Top 10을 알려줘"

### 10. 인코딩 변환기
> "EUC-KR로 인코딩된 파일을 UTF-8로 변환하는 프로그램" (Python 추천)

## 🔴 심화 (배운 것 조합 필요)

### 11. 가사 파일 포맷터

### 12. 파일 비교기
> "두 텍스트 파일을 비교해서 어느 줄이 다른지 보여주는 diff 도구"

### 13. 텍스트 암호화/복호화
> "시저 암호(각 글자를 N칸 밀기)로 텍스트를 암호화/복호화하는 프로그램"

### 14. YAML ↔ JSON 변환기
> "YAML 형식과 JSON 형식을 서로 변환하는 도구" (Python 추천)

- `.txt`와 `.md` 파일은 둘 다 텍스트인데, 왜 굳이 마크다운을 쓸까?
- CSV와 JSON 중 어떤 상황에 어떤 것이 더 적합할까?
- 같은 데이터를 다른 형식으로 저장하면 파일 크기가 어떻게 달라질까?
