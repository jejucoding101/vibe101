## 📖 이론 (20분)

### CSS란?

**C**ascading **S**tyle **S**heets — HTML의 **디자인**을 담당합니다.

```css
/* 선택자 { 속성: 값; } */
h1 {
    color: #2196F3;      /* 글자 색상 */
    font-size: 32px;     /* 글자 크기 */
    text-align: center;  /* 가운데 정렬 */
}
```

### 선택자 3가지

| 선택자 | 문법 | 설명 |
|--------|------|------|
| 태그 | `h1 { }` | 모든 h1 태그에 적용 |
| 클래스 | `.card { }` | `class="card"`인 요소에 적용 |
| ID | `#header { }` | `id="header"`인 요소에 적용 |

### 박스 모델

모든 HTML 요소는 **박스** 형태입니다:

```
┌─────────── margin (바깥 여백) ──────────┐
│ ┌─────── border (테두리) ─────────────┐ │
│ │ ┌────── padding (안쪽 여백) ──────┐ │ │
│ │ │        content (내용)          │ │ │
│ │ └────────────────────────────────┘ │ │
│ └────────────────────────────────────┘ │
└────────────────────────────────────────┘
```

### Flexbox: 배치의 핵심

```css
.container {
    display: flex;           /* 플렉스 활성화 */
    justify-content: center; /* 가로 가운데 */
    align-items: center;     /* 세로 가운데 */
    gap: 16px;               /* 간격 */
}
```

## 🔨 가이드 실습 (25분)

### 실습 1: 13강 페이지에 스타일 입히기 (10분)

```
13강에서 만든 자기소개 HTML에 세련된 CSS를 적용해줘:
- 다크 모드 배경 (#1a1a2e) + 밝은 텍스트
- 구글 폰트 (Noto Sans KR) 적용
- 카드 UI로 섹션 구분 (둥근 모서리, 그림자)
- 프로필 사진 원형 처리
- 링크에 호버 효과 (색상 변경 + 밑줄 애니메이션)
```

### 실습 2: 카드 UI 만들기 (10분)

```
상품 카드 3개를 나란히 배치하는 페이지:
- 각 카드: 이미지, 상품명, 가격, "장바구니" 버튼
- Flexbox로 가로 배치
- 카드에 그림자 + 호버 시 살짝 떠오르는 효과
- 버튼은 그라디언트 배경
```

### 실습 3: 색상 실험 (5분)

```
같은 카드 UI를:
1. 밝은 테마 (흰색 배경)
2. 다크 테마 (어두운 배경)
3. 글래스모피즘 (반투명 유리 효과)
세 가지 버전으로 만들어줘. 비교하고 싶어.
```
