# 📌 19강: 종합 대시보드 — Part 3 총정리

> **핵심 포인트**: HTML/CSS/JS로 배운 모든 것을 **컴포넌트 기반**으로 조합하여 인터랙티브 대시보드 완성

---

## 📖 이론 (20분)

### 컴포넌트 설계 사고방식

하나의 큰 페이지를 작은 **독립 단위(컴포넌트)**로 나눕니다:

```
대시보드 = 헤더 + 사이드바 + 위젯A + 위젯B + 위젯C + 푸터
```

### 파일 분리 원칙

```
dashboard/
├── index.html          ← 전체 구조만
├── css/
│   ├── main.css        ← 공통 스타일
│   └── widgets.css     ← 위젯별 스타일
└── js/
    ├── app.js           ← 초기화 + 이벤트 연결
    ├── clock-widget.js  ← 시계 위젯
    ├── todo-widget.js   ← 투두 위젯
    └── memo-widget.js   ← 메모 위젯
```

> 각 파일이 하나의 역할만 → 수정과 확장이 쉬움!

---

## 🔨 가이드 실습 (25분)

Antigravity에게 단계별로 요청합니다:

### Step 1: 레이아웃 뼈대 (5분)
```
개인 대시보드의 레이아웃을 만들어줘:
- 상단: 로고 + 현재 시각 + 다크모드 토글
- 메인: CSS Grid로 위젯 영역 (2x2)
- 반응형: 모바일에서 1열
- 다크 테마 기본
```

### Step 2: 위젯 3개 구현 (12분)
```
대시보드에 3개 위젯을 추가해줘:

1. ⏰ 시계 위젯: 현재 시각 실시간 표시 + 날짜
2. ✅ 투두 위젯: 할일 추가/완료/삭제 + localStorage 저장
3. 📝 메모 위젯: 간단한 메모 작성 + 자동 저장

각 위젯은 별도 JS 파일로 분리해줘.
```

### Step 3: 폴리싱 (8분)
```
대시보드에 마무리 작업:
- 위젯 로드 시 페이드인 애니메이션
- 버튼 호버 마이크로 인터랙션
- 위젯 접기/펼치기 토글
- 기분 좋은 그라디언트 포인트 색상
```
