# 🎯 화두 풀 — 종합 대시보드

---

## 🟢 입문
1. **날씨 위젯** — "현재 날씨 아이콘 + 온도 표시 (하드코딩 데이터)"
2. **D-day 위젯** — "중요 이벤트까지 남은 일수 카운트다운"
3. **명언 위젯** — "새로고침마다 랜덤 명언 표시"
4. **달력 위젯** — "이번 달 달력을 표 형태로 표시"
5. **북마크 위젯** — "자주 가는 사이트 링크 모음"

## 🟡 도전
6. **포모도로 타이머** ⭐ — "25분 집중 + 5분 휴식 반복 타이머"
7. **습관 트래커** ⭐ — "매일 체크하는 습관 목록 + 주간 달성률 시각화"
8. **칸반 보드** — "To Do / Doing / Done 3열 카드 보드 + 드래그 이동"
9. **운동 기록** — "운동 종류, 시간, 세트 입력 + 주간 차트"
10. **미니 SNS 피드** — "짧은 글 작성 + 타임라인 표시 + 좋아요"

## 🔴 심화
11. **가계부 위젯** — "수입/지출 입력 + 잔액 표시 + 카테고리별 파이차트"
12. **뉴스 피드 리더** — "RSS나 하드코딩 뉴스를 카드 형태로 표시"
13. **드래그 앤 드롭 위젯 재배치** — "위젯 위치를 드래그로 변경 + 저장"

## 💡 실생활
14. **아침 루틴 대시보드** — "시계 + 오늘 날씨 + 할일 + 명언을 한 화면에"
15. **학습 대시보드** — "과목별 학습 시간 + 진도율 + 다음 시험 D-day"
