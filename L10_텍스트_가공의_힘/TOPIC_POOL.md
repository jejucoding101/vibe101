# 🎯 화두 풀 — 텍스트 가공의 힘

## 🟢 입문

### 1. 비속어 필터 ⭐ 추천
> "금지 단어 목록을 정하고, 입력 텍스트에서 해당 단어를 ***로 치환하는 필터"

### 2. 전화번호 포맷 통일
> "다양한 형식의 전화번호를 010-XXXX-XXXX 형태로 일괄 변환"

### 3. URL에서 정보 추출
> "URL을 입력하면 프로토콜, 도메인, 경로, 쿼리 파라미터를 분리하여 보여줘"

### 4. 글자 수 카운터
> "텍스트 입력 → 글자/단어/문장/줄 수를 실시간 계산하는 도구"

### 5. 한영 변환기 (간이)
> "qwerty 자판 기준 한영 변환 매핑 테이블로 변환 (예: gksrmf → 한글)"

## 🟡 도전

### 6. 로그 파일 에러 추출 ⭐ 추천
> "서버 로그 파일에서 ERROR, WARNING 패턴의 줄만 추출하여 리포트 생성" (Python 추천)

### 7. 마크다운 → HTML 변환기
> "간단한 마크다운(제목, 볼드, 리스트)을 HTML로 변환하는 도구"

### 8. 해시태그 추출기
> "SNS 게시글에서 #해시태그만 추출하여 빈도순으로 정렬"

### 9. CSV 필드 정규화
> "CSV의 이름 필드에서 공백 정리, 대소문자 통일, 특수문자 제거"

### 10. 주민번호/카드번호 마스킹
> "텍스트에서 주민번호, 카드번호 패턴을 찾아 중간 6자리를 ******로 마스킹"

## 🔴 심화

### 11. 형태소 분리기 (간이)
> "한국어 문장에서 조사(은/는/이/가/을/를)를 분리하는 간이 형태소 분석기" (Python 추천)

### 12. 자동 요약기 (간이)
> "긴 텍스트에서 가장 중요한 문장 3개를 추출하는 간이 요약 도구"

### 13. 정규식 테스터
> "정규식 패턴을 입력하면 테스트 문자열에서 매칭 결과를 보여주는 도구"
