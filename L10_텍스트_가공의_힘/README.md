> **핵심 포인트**: `split`, `join`, `replace`, 정규식 기초로 텍스트 데이터를 자유롭게 가공하기

---

## 📖 이론 (20분)

### 문자열이 중요한 이유

프로그래밍에서 다루는 데이터의 대부분은 **텍스트(문자열)**입니다. 이메일, 전화번호, URL, 로그, 채팅, 문서... 텍스트를 자유롭게 가공할 수 있으면 자동화의 범위가 크게 넓어집니다.

### 핵심 메서드 6개

| 메서드 | 하는 일 | 예시 |
|--------|---------|------|
| `split()` | 문자열 쪼개기 | `"a,b,c".split(",")` → `["a","b","c"]` |
| `join()` | 배열 합치기 | `["a","b"].join("-")` → `"a-b"` |
| `replace()` | 치환하기 | `"hello".replace("l","r")` → `"herlo"` |
| `trim()` / `strip()` | 양쪽 공백 제거 | `" hi ".trim()` → `"hi"` |
| `includes()` / `in` | 포함 여부 | `"hello".includes("ell")` → `true` |
| `toUpperCase()` / `upper()` | 대소문자 변환 | `"hi".toUpperCase()` → `"HI"` |

### 정규식 맛보기

**정규식(Regular Expression)**: 텍스트 패턴을 표현하는 특수 문법

```
이메일: \w+@\w+\.\w+
전화번호: \d{3}-\d{4}-\d{4}
숫자만: \d+
한글만: [가-힣]+
```

> 💡 정규식을 외울 필요 없습니다. AI에게 "이런 패턴을 찾는 정규식을 만들어줘"라고 하면 됩니다!
