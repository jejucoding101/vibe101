# 🎯 화두 풀 — 서버 만들기

---

## 🟢 입문
1. **인사 API** — "이름을 보내면 '안녕, {이름}!'을 반환하는 API"
2. **시간 API** — "세계 시간대별 현재 시각을 반환"
3. **계산기 API** ⭐ — "GET /calc?a=10&b=5&op=add → 결과 반환"
4. **명언 API** — "카테고리별 랜덤 명언 반환"
5. **사전 API** — "단어 입력 → 정의 반환 (하드코딩 사전)"

## 🟡 도전
6. **단축 URL 서비스** ⭐ — "긴 URL → 짧은 코드 생성 + 리다이렉트"
7. **QR 코드 생성** — "텍스트 입력 → QR 코드 이미지 반환" (Python 추천)
8. **마크다운→HTML API** — "마크다운 텍스트를 HTML로 변환해서 반환"
9. **피드백 수집** — "사용자 피드백을 받아 JSON 파일에 저장하는 API"
10. **간이 챗 서버** — "메시지 전송/수신이 가능한 초간단 채팅 API"

## 🔴 심화
11. **파일 업로드** — "파일을 업로드하면 서버에 저장하고 URL 반환"
12. **이미지 리사이즈** — "이미지 URL + 크기 → 리사이즈된 이미지 반환" (Python 추천)

## 💡 실생활
13. **팀 공유 메모** — "팀원들이 같은 메모장을 사용하는 간이 서비스"
14. **알림 API** — "정해진 시간에 알림 메시지를 반환하는 API"
