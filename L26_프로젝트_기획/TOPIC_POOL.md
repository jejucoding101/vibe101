# 🎯 화두 풀 — 프로젝트 기획

> 아래 프로젝트 아이디어 중 하나를 골라 기획서를 작성해보세요!

---

## 프로젝트 아이디어 모음

### 📋 생산성
1. **습관 트래커** ⭐ — 매일 습관 체크, 연속 달성, 통계 대시보드
2. **포모도로 타이머 앱** — 집중/휴식 타이머 + 일간 실적 기록
3. **칸반 보드** — To Do / Doing / Done 카드 보드
4. **회의록 관리** — 회의 기록, 참석자, 액션 아이템 관리
5. **프로젝트 관리 도구** — 프로젝트별 할일, 마감일, 진행률

### 📝 콘텐츠
6. **블로그/일기장** ⭐ — 글 작성, 태그, 검색, 공개/비공개 설정
7. **레시피 북** — 레시피 CRUD, 재료 기반 검색, 요리 시간 표시
8. **독서 노트** — 책별 메모, 인용구, 평점, 통계
9. **미니 위키** — 마크다운 문서, 내부 링크, 카테고리
10. **학습 노트** — 과목별 노트, 플래시카드, 퀴즈 출제

### 🛒 비즈니스
11. **상품+장바구니** ⭐ — 상품 목록, 카테고리, 장바구니, 주문 내역
12. **URL 북마크 관리** — 링크 저장, 태그 분류, 검색, 공유
13. **이벤트 관리** — 이벤트 생성, 참가자 RSVP, 일정 캘린더
14. **포트폴리오 빌더** — 프로젝트 소개, 기술 스택, 연락처

### 🎮 재미
15. **플래시카드 퀴즈** — 질문/답 카드, 랜덤 출제, 정답률 통계
16. **미니 SNS** — 짧은 글 작성, 타임라인, 좋아요, 프로필

## 기획서 작성 가이드

Antigravity에게 이렇게 요청하면 좋습니다:

```
[프로젝트명]의 기획서를 작성해줘.
다음 항목을 포함해줘:
1. 프로젝트 개요 (한 줄)
2. 대상 사용자
3. MVP 기능 목록 (필수 5~7개)
4. 추후 확장 기능 (3~5개)
5. DB 테이블 설계 (SQL)
6. API 엔드포인트 목록
7. 구현 순서 로드맵
```
