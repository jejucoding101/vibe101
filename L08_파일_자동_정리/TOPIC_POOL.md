# 🎯 화두 풀 — 파일 자동 정리

---

## 🟢 입문

### 1. 사진 날짜별 정리 ⭐ 추천
> "사진 파일을 수정일 기준 YYYY-MM 폴더로 자동 분류"

### 2. 빈 폴더 찾아 삭제
> "하위 폴더까지 훑어서 빈 폴더를 찾고, 확인 후 삭제하는 도구"

### 3. 특정 확장자 일괄 삭제
> ".tmp, .log 파일만 찾아서 삭제하는 정리 도구" (삭제 전 확인!)

### 4. 파일 크기 리포트
> "폴더 안 파일을 크기순으로 정렬하여 Top 20 리포트 생성"

### 5. 파일명에 번호 매기기
> "001_파일명.확장자 형태로 일련번호를 자동으로 붙이는 도구"

## 🟡 도전

### 6. 중복 파일 찾기 ⭐ 추천
> "같은 이름 또는 같은 크기의 파일을 찾아 중복 후보 리스트 생성" (Python 추천)

### 7. 자동 백업 도구
> "중요 폴더를 backup_YYYYMMDD 이름으로 자동 복사하는 백업 스크립트"

### 8. 워터마크 파일명 추가
> "이미지 파일명에 '[WM]' 접두사를 붙이거나 제거하는 토글 도구"

### 9. 압축/해제 자동화 (Python 추천)
> "특정 폴더를 ZIP으로 압축하거나, ZIP 파일을 해제하는 도구"

### 10. 로그 로테이션
> "로그 파일이 10MB를 넘으면 자동으로 이름 변경(날짜 추가)하고 새 파일 시작"

## 🔴 심화

### 11. 디렉토리 동기화
> "소스 폴더의 변경사항을 대상 폴더에 반영하는 단방향 동기화 도구"

### 12. 파일 감시자
> "폴더를 모니터링하다가 새 파일이 추가되면 자동으로 분류 실행"

### 13. 용량 분석 리포트
> "폴더별 용량을 계산하여 마크다운 리포트로 생성 (트리 형태)"

## 💡 실생활 응용

### 14. 프로젝트 폴더 템플릿
> "새 프로젝트를 시작할 때 표준 폴더 구조를 자동 생성하는 초기화 도구"

### 15. 다운로드 자동 정리
> "다운로드 폴더를 주기적으로 확인하여 자동 분류하는 상시 실행 스크립트"

## 🤔 생각해볼 점

- 실수로 중요한 파일을 삭제하면? → "휴지통으로 이동" 방식의 안전장치
- dry-run(미리보기) 없이 바로 파일을 옮기면 어떤 위험이 있을까?
- 이런 자동화 스크립트를 "매일 자동 실행"하려면? → 30강 예고
