# 📌 13강: HTML 기초 — 웹의 뼈대

> **핵심 포인트**: HTML 태그, 문서 구조(head/body), 시맨틱 태그로 웹 페이지의 구조를 만들기

---

## 📖 이론 (20분)

### 웹 페이지는 3가지로 구성됩니다

| 역할 | 기술 | 비유 |
|------|------|------|
| **구조** (뼈대) | HTML | 건물의 골조 |
| **디자인** (외관) | CSS | 인테리어, 페인트 |
| **동작** (기능) | JavaScript | 전기, 엘리베이터 |

### HTML 태그란?

`<태그명>내용</태그명>` 형태로, 각 태그가 역할을 가집니다:

```html
<h1>큰 제목</h1>          <!-- 제목 (h1~h6) -->
<p>문단 텍스트</p>         <!-- 문단 -->
<a href="url">링크</a>    <!-- 하이퍼링크 -->
<img src="사진.jpg">       <!-- 이미지 -->
<ul><li>목록</li></ul>     <!-- 순서 없는 목록 -->
<button>클릭!</button>     <!-- 버튼 -->
```

### HTML 문서 기본 구조

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>페이지 제목</title>
</head>
<body>
    <!-- 여기에 보이는 내용 -->
    <h1>안녕하세요!</h1>
</body>
</html>
```

> 💡 모든 것을 외울 필요 없습니다. AI에게 "이런 페이지 만들어줘"라고 하면 적절한 태그를 자동으로 사용합니다. 우리는 **결과를 보고 이해**하면 됩니다.

## 🔨 가이드 실습 (25분)

### 실습 1: 자기소개 페이지 (10분)

```
나를 소개하는 웹 페이지를 HTML로 만들어줘:
- 이름, 직업, 한 줄 소개
- 취미 목록 (순서 없는 목록)
- 좋아하는 것 TOP 3 (순서 있는 목록)
- 연락처 (이메일 링크)
- 프로필 placeholder 이미지
파일명: index.html
```

브라우저에서 열어보세요! (파일 더블클릭)

### 실습 2: 태그 익히기 (10분)

```
다음 태그를 모두 사용한 예제 HTML 페이지를 만들어줘:
- 표(table): 이번 주 시간표
- 폼(form): 간단한 피드백 양식 (이름, 이메일, 메시지, 제출 버튼)
- 구분선(hr)과 인용문(blockquote)
- 시맨틱 태그: header, nav, main, footer
각 태그 옆에 주석으로 "이 태그는 ~입니다" 설명 추가
```

### 실습 3: 내용 수정 체험 (5분)

AI가 만든 HTML에서 직접 수정해보세요:
- 텍스트 내용 변경
- 이미지 URL 변경
- 링크 추가
- 수정 후 브라우저 새로고침으로 즉시 확인!
