이론

CSV란 무엇인가요?

CSV는 Comma-Separated Values의 약자로, 가장 단순한 표(table) 형식의 데이터 파일입니다. 엑셀 파일(.xlsx)과 달리 메모장으로도 열어볼 수 있고, 어떤 프로그래밍 언어에서든 쉽게 읽고 쓸 수 있습니다.

이름나이도시
홍길동25서울
김철수30부산
이영희28대전
첫 줄 = 헤더(열 이름)
나머지 줄 = 행(데이터 레코드)
쉼표(,) = 열 구분자
  data.csv 
← 헤더 행 (열 이름을 정의합니다)
이름,나이,도시

CSV는 어디에서 사용되나요?

CSV는 서로 다른 시스템 간에 표 형식 데이터를 주고받을 때 가장 널리 쓰이는 형식입니다. 엑셀, 구글 시트, 데이터베이스, 웹 서비스 등 거의 모든 도구가 CSV를 지원합니다.

JavaScript vs Python — CSV 처리 비교

두 언어 모두 CSV를 다룰 수 있지만, 접근 방식이 다릅니다. JavaScript는 파일을 텍스트로 읽은 뒤 직접 파싱해야 하고, Python은 내장 csv 모듈이 파싱을 자동으로 처리해 줍니다.

  JavaScript
// fs 모듈로 파일을 읽고
// 직접 줄바꿈과 쉼표로 나눕니다
const fs = require('fs');
const data = fs.readFileSync(
  'data.csv', 'utf8'
);
const rows = data
  .split('\n')
  .map(row => row.split(','));
  Python
# 내장 csv 모듈이 파싱을
# 자동으로 처리해 줍니다
import csv

with open('data.csv', 'r',
     encoding='utf-8') as f:
    reader = csv.DictReader(f)
    for row in reader:
        print(row['이름'], row['나이'])
💡
CSV 처리는 Python이 더 편리합니다. 내장 csv 모듈의 DictReader를 사용하면 각 행을 딕셔너리(사전) 형태로 바로 접근할 수 있어서 코드가 훨씬 직관적입니다.

실습 1: CSV 파일 생성 8분

AI에게 아래 프롬프트를 입력하여 학생 성적 데이터 CSV를 자동으로 생성합니다.

학생 10명의 성적 데이터를 CSV 파일로 만들어줘:
- 열: 이름, 국어, 영어, 수학, 과학
- 점수는 50~100 사이 랜덤으로
- 파일명: students.csv
- JavaScript와 Python 둘 다

실습 2: CSV 읽기 + 필터링 10분

생성한 CSV를 읽어서 조건에 맞는 데이터를 추출하고, 통계를 계산합니다.

students.csv를 읽어서:
1. 전체 학생 목록을 보기 좋은 표로 출력
2. 국어 점수 80점 이상인 학생만 필터링
3. 각 과목별 평균 점수 계산
4. 결과를 콘솔에 출력하고, honors.csv로도 저장

실습 3: CSV 수정 7분

기존 CSV에 새로운 계산 열을 추가하여 가공된 데이터를 새 파일로 저장합니다.

students.csv에 새 열 '평균'과 '등급'을 추가해줘:
- 평균 = 4과목 합계 / 4
- 등급 = A(90↑), B(80↑), C(70↑), D(60↑), F(60↓)
- 수정된 결과를 students_final.csv로 저장
