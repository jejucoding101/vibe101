# 📌 9강: 반복과 조건 — 프로그래밍의 핵심 제어 구조

> **핵심 포인트**: `for`, `while`, `if/else` — AI가 만든 코드를 이해하고 수정하기 위해 반드시 알아야 할 것

---

## 📖 이론 (20분)

### 왜 반복과 조건을 알아야 할까?

바이브코딩에서 코드를 직접 쓰지는 않지만, AI가 만든 코드를 **읽고 이해하고 수정 요청**하려면 핵심 구조를 알아야 합니다. 그 핵심이 바로 **반복문**과 **조건문**입니다.

### 조건문 (if / else)

"만약 ~라면, ~해라"

```javascript
// JavaScript
if (score >= 90) {
    console.log("A등급");
} else if (score >= 80) {
    console.log("B등급");
} else {
    console.log("C등급");
}
```

```python
# Python
if score >= 90:
    print("A등급")
elif score >= 80:
    print("B등급")
else:
    print("C등급")
```

### 반복문 (for / while)

"이 작업을 N번 반복해라"

```javascript
// JavaScript — 배열의 각 항목에 대해
for (const name of names) {
    console.log(`안녕, ${name}!`);
}
```

```python
# Python — 리스트의 각 항목에 대해
for name in names:
    print(f"안녕, {name}!")
```

### 핵심 패턴 3가지

| 패턴 | 설명 | 예시 |
|------|------|------|
| **순회** | 목록의 각 항목에 작업 적용 | 파일 목록에서 하나씩 처리 |
| **필터** | 조건에 맞는 것만 선택 | 80점 이상만 추출 |
| **누적** | 반복하며 값을 쌓아감 | 합계 구하기, 카운트 |

## 🔨 가이드 실습 (25분)

### 실습 1: 반복문 체험 (8분)

```
1부터 100까지 숫자 중:
- 3의 배수이면 "Fizz"
- 5의 배수이면 "Buzz"  
- 3과 5의 공배수이면 "FizzBuzz"
- 나머지는 숫자 그대로 출력
하는 프로그램을 만들어줘. (유명한 FizzBuzz 문제!)
JS와 Python 둘 다.
```

AI가 만든 코드에서 **for문과 if/else문**을 찾아 표시해보세요.

### 실습 2: 실생활 데이터에 적용 (10분)

```
학생 성적 JSON 데이터를 만들고:
1. 전체 학생을 순회하며 이름과 평균 출력 (반복)
2. 평균 80점 이상인 학생만 우등생 목록에 추가 (조건 + 필터)
3. 전체 평균, 최고 점수, 최저 점수 계산 (누적)
```

### 실습 3: 코드 수정 체험 (7분)

실습 2의 결과 코드에서 AI에게:
```
조건을 바꿔줘:
- 우등생 기준을 90점으로 변경
- 60점 미만은 "재시험 대상"으로 별도 분류
- 결과를 JSON 파일로 저장
```

> 🎵 코드를 읽을 수 있으니, "어디를 어떻게 바꿔달라"고 더 정확하게 요청할 수 있습니다!
