데이터 시각화

숫자를 차트로 보여주기 — Chart.js와 Matplotlib로 데이터를 시각적으로 표현하는 방법을 배웁니다.

📖

이론

⏱ 20분

왜 시각화가 중요한가요?

같은 데이터라도 표와 차트는 전달력에서 큰 차이를 보입니다. 표는 "매출이 1월 100, 2월 120, 3월 80..."처럼 숫자의 나열이지만, 차트는 한눈에 추이와 비교를 가능하게 합니다. 데이터의 패턴, 이상치, 트렌드를 직관적으로 파악하려면 시각화가 필수입니다.

차트 종류와 용도

데이터의 성격에 따라 적합한 차트가 다릅니다. 아래 5가지가 가장 기본적이며 자주 사용되는 차트입니다.

JavaScript vs Python — 시각화 도구 비교

JavaScript는 Chart.js로 웹 브라우저에서 인터랙티브 차트를 만들고, Python은 Matplotlib로 이미지 파일을 생성합니다. 두 도구의 특성은 아래와 같습니다.

⚡ Chart.js (JavaScript)
  • 웹페이지(HTML)에 바로 표시됩니다.
  • 인터랙티브: 마우스 오버, 클릭 이벤트를 지원합니다.
  • CDN으로 설치 없이 바로 사용할 수 있습니다.
  • 실시간 데이터 업데이트에 적합합니다.
🐍 Matplotlib (Python)
  • 이미지 파일(PNG, SVG)로 저장됩니다.
  • 정적 차트: 보고서, 논문에 적합합니다.
  • pip install matplotlib로 설치합니다.
  • 데이터 분석(Pandas)과 연동이 뛰어납니다.
💡
어떤 도구를 선택할지 고민된다면, 웹에서 보여줄 차트는 Chart.js, 파일로 저장하거나 분석할 차트는 Matplotlib를 사용하세요. 이 강의에서는 두 도구를 모두 실습합니다.
🔨

가이드 실습

⏱ 25분

실습 1: 첫 차트 만들기 10분

과일별 판매량 데이터로 막대 그래프를 만들어봅니다. JavaScript와 Python 양쪽 모두에서 동일한 데이터를 시각화합니다.

과일별 판매량 데이터로 막대 그래프를 만들어줘:
- 사과: 120, 바나나: 85, 오렌지: 70, 포도: 95, 딸기: 150
- JavaScript 버전: Chart.js로 HTML 페이지
- Python 버전: matplotlib로 이미지 파일 저장

실습 2: CSV 데이터 → 차트 10분

6강에서 만든 학생 성적 CSV를 읽어서 의미 있는 차트로 변환합니다. 데이터를 수집하고 시각화하는 전체 흐름을 경험합니다.

6강에서 만든 학생 성적 CSV를 읽어서:
1. 과목별 평균 점수 → 막대 그래프
2. 등급별(A/B/C/D/F) 학생 수 → 파이 차트
JS와 Python 둘 다 만들어줘.

실습 3: 인터랙티브 차트 5분

Chart.js의 인터랙티브 기능을 추가하여 사용자가 차트와 상호작용할 수 있도록 합니다. 정적 차트와의 차이를 체감합니다.

JS Chart.js 버전에 인터랙티브 기능을 추가해줘:
- 마우스 오버 시 수치 표시
- 범례 클릭으로 데이터 숨기기/보이기
- 막대/선/파이 차트 전환 버튼

🟢 입문
일주일 공부 시간
요일별 공부 시간을 막대 그래프로 표시하고, 목표 대비 달성률을 함께 보여줍니다.

이번 강의 체크리스트

  • 데이터에 적합한 차트 종류를 선택할 수 있습니다.
  • Chart.js(JS) 또는 Matplotlib(Python)로 차트를 생성할 수 있습니다.
  • CSV 또는 JSON 데이터를 읽어서 차트로 변환할 수 있습니다.
  • 차트의 색상, 제목, 범례 등을 커스터마이징할 수 있습니다.
