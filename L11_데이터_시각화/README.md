## 📖 이론 (20분)

### 왜 시각화가 중요한가?

같은 데이터라도 **표 vs 차트**의 이해도 차이는 큽니다:
- 표: "매출이 1월 100, 2월 120, 3월 80..." → 숫자의 나열
- 차트: 한눈에 추이와 비교가 가능!

### 차트 종류와 용도

| 차트 | 용도 | 예시 |
|------|------|------|
| **막대(Bar)** | 항목 간 크기 비교 | 과목별 평균 점수 |
| **선(Line)** | 시간에 따른 추이 | 월별 매출 변화 |
| **파이(Pie)** | 전체 대비 비율 | 혈액형 분포 |
| **도넛(Doughnut)** | 비율 + 중앙 여백 | 카테고리별 지출 |
| **레이더(Radar)** | 다항목 비교 | 게임 캐릭터 스탯 |

### JS vs Python 도구

| | JavaScript | Python |
|---|-----------|--------|
| **라이브러리** | Chart.js | matplotlib |
| **출력 형태** | 웹페이지(HTML) | 이미지 파일(PNG) |
| **인터랙티브** | ✅ (마우스 오버 등) | 제한적 |
| **설치** | CDN으로 바로 사용 | `pip install matplotlib` |

## 🔨 가이드 실습 (25분)

### 실습 1: 첫 차트 만들기 (10분)

```
과일별 판매량 데이터로 막대 그래프를 만들어줘:
- 사과: 120, 바나나: 85, 오렌지: 70, 포도: 95, 딸기: 150
- JavaScript 버전: Chart.js로 HTML 페이지
- Python 버전: matplotlib로 이미지 파일 저장
```

### 실습 2: CSV 데이터 → 차트 (10분)

```
6강에서 만든 학생 성적 CSV를 읽어서:
1. 과목별 평균 점수 → 막대 그래프
2. 등급별(A/B/C/D/F) 학생 수 → 파이 차트
JS와 Python 둘 다 만들어줘.
```

### 실습 3: 인터랙티브 차트 (5분)

```
JS Chart.js 버전에 인터랙티브 기능을 추가해줘:
- 마우스 오버 시 수치 표시
- 범례 클릭으로 데이터 숨기기/보이기
- 막대/선/파이 차트 전환 버튼
```
