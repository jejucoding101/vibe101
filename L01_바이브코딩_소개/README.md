# 📌 1강: 바이브코딩 소개 — AI에게 일 시키기

> **핵심 포인트**: AI 에이전트(Antigravity)에게 자연어로 지시하여 프로그램을 만드는 바이브코딩의 개념과 첫 경험

---

## 📖 이론 (20분)

### 바이브코딩이란?

2025년 초, AI 연구자 **Andrej Karpathy**가 처음 제안한 개념입니다.

> "더 이상 코드를 직접 작성하지 않습니다. AI에게 무엇을 만들라고 '말'하면, AI가 코드를 작성합니다. 저는 결과를 보고 '느낌(vibes)'으로 방향을 잡습니다."

#### 전통 코딩 vs 바이브코딩

| | 전통 코딩 | 바이브코딩 |
|---|-----------|-----------|
| **입력** | 프로그래밍 언어 문법 | 자연어 (한국어/영어) |
| **주체** | 개발자가 코드 작성 | AI가 코드 작성 |
| **역할** | 타이피스트 + 설계자 | 설계자 + 검증자 |
| **필수 스킬** | 문법 암기, 디버깅 | 명확한 요구사항 정리, 결과 검증 |
| **진입 장벽** | 높음 (수개월~수년) | 낮음 (오늘부터 가능) |

#### Antigravity란?

Google DeepMind가 만든 **AI 코딩 에이전트**입니다.

- 파일을 읽고, 만들고, 수정할 수 있습니다
- 터미널 명령어를 실행할 수 있습니다
- 웹을 검색하고 정보를 가져올 수 있습니다
- 질문하면 코드를 설명해줍니다
- **한국어로 대화** 가능합니다

#### 바이브코딩의 3단계

```
1. 생각한다 (Think)  → "무엇을 만들까?"
2. 지시한다 (Prompt)  → "이렇게 만들어줘"
3. 검증한다 (Verify)  → "잘 작동하는지 확인"
```

### JavaScript와 Python

이 강좌에서는 두 가지 언어를 병행합니다:

| | JavaScript | Python |
|---|-----------|--------|
| **특징** | 웹 브라우저 + 서버 | 범용 + 데이터 분석 |
| **실행** | `node 파일명.js` | `python 파일명.py` |
| **파일 확장자** | `.js` | `.py` |
| **강점 분야** | 웹 앱, 인터랙티브 UI | 파일 처리, 자동화, 데이터 |

> 💡 어떤 언어를 써야 할지 모르겠으면? 걱정 마세요. Antigravity에게 "이 작업은 무슨 언어가 좋아?"라고 물어보면 됩니다!

## 🔨 가이드 실습 (25분)

### 실습 1: 첫 프로그램 만들기 (10분)

Antigravity에게 다음과 같이 요청해보세요:

```
"안녕하세요!"를 출력하는 프로그램을 만들어줘.
JavaScript 버전과 Python 버전 둘 다 만들어줘.
```

**기대 결과**: `hello.js`와 `hello.py` 파일이 생성됩니다.

AI가 만든 파일을 확인해보세요:
- 어떤 폴더에 저장되었나요?
- 파일 내용은 어떻게 생겼나요?
- 어떻게 실행하라고 안내했나요?

### 실습 2: 조금 더 복잡한 것 (10분)

```
사용자에게 이름을 물어보고, "안녕하세요, [이름]님! 오늘 날짜는 [날짜]입니다."
라고 출력하는 프로그램을 만들어줘. JS와 Python 둘 다.
```

실행해보고 관찰해보세요:
- 프로그램이 나에게 질문을 합니다 (입력 대기)
- 내가 입력하면 결과가 나옵니다
- 날짜는 어디서 가져올까요? → AI가 날짜 관련 기능을 알아서 사용합니다

### 실습 3: AI와 대화하며 수정하기 (5분)

방금 만든 프로그램을 수정해봅시다:

```
방금 만든 인사 프로그램을 수정해줘.
시간대에 따라 "좋은 아침이에요", "좋은 오후에요", "좋은 저녁이에요"로 바꿔줘.
```

> 🎵 이것이 바이브코딩입니다! 코드를 한 줄도 직접 쓰지 않았지만, 프로그램이 점점 똑똑해지고 있습니다.
