# 📊 공개 데이터 소스 가이드

> 강의에서 활용할 수 있는 고품질 CSV/JSON 데이터를 무료로 얻을 수 있는 사이트 모음입니다.

---

## 🇰🇷 한글 데이터 소스

### 1. 공공데이터포털 ⭐ 최우선 추천
- **URL**: https://data.go.kr
- **운영**: 대한민국 정부
- **특징**: 국내 최대 통합 공공데이터 플랫폼. 파일 데이터(CSV/XLS) 직접 다운로드 가능
- **주제**: 교통, 인구, 관광, 보건, 환경, 경제, 행정 등 거의 모든 분야
- **팁**: `데이터 찾기` → `파일 데이터` 유형 선택 → `CSV` 확장자 필터
- **⚠️ 주의**: 일부 파일이 EUC-KR 인코딩 → Python에서 `encoding='euc-kr'` 필요

### 2. 서울 열린데이터광장 ⭐ 교육 최적
- **URL**: https://data.seoul.go.kr
- **운영**: 서울특별시
- **특징**: 서울시 행정 데이터가 매우 상세하게 정리됨. CSV 직접 다운로드
- **주제**: 대중교통 이용량, 구별 인구, 공공 와이파이, 문화시설, 생활물가
- **장점**: 데이터가 깨끗하고 칼럼 설명이 잘 되어 있음

### 3. 국가통계포털 (KOSIS)
- **URL**: https://kosis.kr
- **운영**: 통계청
- **특징**: 신뢰도 최고의 국가 공식 통계. CSV/엑셀 다운로드 가능
- **주제**: 경제 지표, 인구 통계, 고용, 물가, 무역
- **팁**: 표 조회 후 `다운로드` 버튼 → CSV 선택

### 4. 경기데이터드림
- **URL**: https://data.gg.go.kr
- **운영**: 경기도
- **특징**: 경기도 공공 데이터. 지도 기반 시각화 자료도 풍부
- **주제**: 경기도 지역 행정, 교통, 복지

### 5. 문화데이터광장
- **URL**: https://culture.go.kr
- **운영**: 문화체육관광부
- **특징**: 문화예술·관광·도서 분야 특화 데이터
- **주제**: 관광지 방문자 수, 도서관 현황, 문화행사 일정

## 🌍 영어 데이터 소스

### 1. Maven Analytics Data Playground ⭐ 교육 최적
- **URL**: https://mavenanalytics.io/data-playground
- **특징**: 분석 연습 목적으로 정제된 깨끗한 데이터. 로그인 없이 다운로드
- **주제**: 커피숍 매출, 영화 평점, 항공편, 피자 판매, 게임 등
- **장점**: 소규모(100~1000행), 칼럼 설명 포함, 바로 분석 가능

### 2. Kaggle Datasets ⭐ 최대 규모
- **URL**: https://kaggle.com/datasets
- **특징**: 커뮤니티 기여 수만 개 데이터. 주제 무한
- **주제**: Titanic, Spotify, Netflix, 날씨, 주식, 건강 등 모든 분야
- **⚠️ 주의**: 로그인(무료 가입) 필수. 일부 데이터셋은 대용량

### 3. FiveThirtyEight Data ⭐ 깨끗한 데이터
- **URL**: https://data.fivethirtyeight.com
- **특징**: 데이터 저널리즘 매체. 기사에 사용된 원본 데이터 공개
- **주제**: 정치, 스포츠(NBA/NFL/축구), 영화 평점, 사회 이슈
- **장점**: 극도로 깨끗하고 맥락이 명확함

### 4. Tidy Tuesday (GitHub)
- **URL**: https://github.com/rfordatascience/tidytuesday
- **특징**: 매주 새 데이터셋 공개. 이미 정제된 상태
- **주제**: 매주 다름 — 환경, 교육, 스포츠, 기술 등
- **장점**: 시각화 연습에 최적

### 5. UCI Machine Learning Repository
- **URL**: https://archive.ics.uci.edu
- **특징**: 가장 오래된(1987~) 데이터 저장소. 교과서적 데이터셋
- **주제**: Iris(붓꽃), Wine(와인), Housing(주택), Heart Disease(심장병)
- **장점**: 데이터 과학의 "hello world" — 모든 튜토리얼에서 사용

### 6. Google Dataset Search
- **URL**: https://datasetsearch.research.google.com
- **특징**: 데이터 전용 검색엔진. 형식별 필터 가능
- **용도**: 특정 주제 데이터를 찾을 때 구글 검색처럼 사용

### 7. World Bank Open Data
- **URL**: https://data.worldbank.org
- **특징**: 세계 경제·개발 지표. 국가별 연도별 데이터
- **주제**: GDP, 인구, 기대수명, 교육, 환경

### 8. Data.gov (미국)
- **URL**: https://data.gov
- **특징**: 미국 정부 공개 데이터
- **주제**: 경제, 건강, 교육, 환경, 기후

### 9. Awesome Public Datasets (GitHub)
- **URL**: https://github.com/awesomedata/awesome-public-datasets
- **특징**: 주제별로 분류된 공개 데이터 링크 모음 (메타 소스)
- **용도**: 원하는 주제의 데이터를 찾는 출발점

## 💡 교육용 데이터 선택 기준

1. **행 수**: 50~500행이 교육에 적합 (너무 작으면 분석 의미 ↓, 너무 크면 로딩·이해 부담 ↑)
2. **칼럼 수**: 4~10개가 이상적
3. **인코딩**: UTF-8 권장 (한글 데이터는 EUC-KR 주의)
4. **결측치**: 초급 강의에서는 결측치 없는 깨끗한 데이터 사용
5. **주제 친숙도**: 학습자가 이미 아는 주제(영화, 음식, 날씨 등) → 데이터 이해가 빠름
