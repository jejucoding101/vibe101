# 🎯 화두 풀 — REST API 설계

---

## 🟢 입문
1. **할일 API** — "할일 CRUD + 완료 토글"
2. **연락처 API** ⭐ — "이름, 전화, 이메일 CRUD"
3. **일기장 API** — "날짜별 일기 CRUD"
4. **북마크 API** — "URL, 제목, 카테고리 CRUD"
5. **명언 API v2** — "CRUD + 카테고리 필터 + 랜덤"

## 🟡 도전
6. **게시판 API** ⭐ — "제목, 내용, 작성자, 날짜 + 페이지네이션(한 페이지 10개씩)"
7. **레시피 API** — "재료(배열), 조리 단계(배열) 포함 복잡한 구조"
8. **설문 API** — "질문 생성 + 응답 수집 + 결과 집계"
9. **일정 API** — "일정 CRUD + 날짜 범위 검색"
10. **상품 관리 API** — "이름, 가격, 재고 CRUD + 가격/카테고리 필터"

## 🔴 심화
11. **투표 API** — "투표 주제 생성 + 투표 + 실시간 결과"
12. **댓글 시스템 API** — "게시글에 댓글 CRUD (중첩 리소스)"
13. **API 문서 자동 생성** — "Swagger/OpenAPI 형식으로 API 문서 자동화"

## 💡 실생활
14. **운동 기록 API** — "종류, 세트, 무게, 시간+날짜별 기록 관리"
15. **독서 목록 API** — "책 CRUD + 읽기 상태(읽음/읽는중/예정) 관리"
