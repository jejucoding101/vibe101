# 🎯 화두 풀 — 풀스택 연동

---

## 🟢 입문
1. **투두 풀스택** ⭐ — "투두 리스트를 서버+DB로 영구 저장"
2. **방명록** ⭐ — "이름+메시지 작성 → DB 저장 → 전체 표시"
3. **카운터 공유** — "여러 사용자가 같은 카운터를 +1/-1 할 수 있는 앱"
4. **명언 표시기** — "DB의 명언을 랜덤으로 프론트에 표시"
5. **간단한 설문** — "질문 표시 → 응답 수집 → 결과 확인"

## 🟡 도전
6. **게시판** — "글 작성/목록/상세보기/삭제가 되는 간단한 게시판"
7. **투표 시스템** — "투표 주제 생성 + 투표 + 실시간 결과 차트"
8. **채팅 앱 (폴링)** — "2초마다 서버에서 새 메시지 확인하는 간이 채팅"
9. **위키 시스템** — "마크다운 문서 작성/편집/목록 관리"
10. **링크 공유** — "URL + 제목 + 태그로 북마크 공유 서비스"

## 🔴 심화
11. **실시간 업데이트** — "Server-Sent Events로 실시간 데이터 갱신"
12. **파일 업로드** — "이미지 업로드 + 서버 저장 + 프론트에서 표시"
13. **페이지네이션** — "메모가 많을 때 페이지 단위로 불러오기"

## 💡 실생활
14. **팀 할일 공유** — "여러 사람이 접속하여 같은 할일 목록 관리"
15. **가계부 웹** — "12강 가계부를 웹 풀스택으로 업그레이드"
