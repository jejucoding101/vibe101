# 🎯 화두 풀 — 로컬 스토리지

---

## 🟢 입문
1. **장바구니 유지** ⭐ — "상품 추가/삭제, 새로고침해도 장바구니 유지"
2. **설정 저장** — "글꼴 크기, 색상 테마, 언어 선택을 localStorage에 저장"
3. **최근 검색어** — "검색 기록 저장 + 목록 표시 + 개별/전체 삭제"
4. **즐겨찾기** — "북마크 추가/삭제, localStorage에 영구 저장"
5. **방문 카운터** — "이 페이지를 몇 번 방문했는지 카운트"

## 🟡 도전
6. **학습 진도 추적** ⭐ — "30강 커리큘럼의 완료 체크를 localStorage에 저장"
7. **게임 세이브** — "간단한 게임(숫자 맞추기)의 최고 점수, 연속 기록 저장"
8. **폼 자동 저장** — "긴 폼 입력 중 실수로 닫아도 입력 내용이 보존"
9. **읽기 진행률** — "긴 글의 스크롤 위치를 저장, 다음 방문 시 이어서"
10. **다국어 설정** — "한국어/영어 UI 전환 + 선택 저장"

## 🔴 심화
11. **데이터 내보내기/가져오기** — "localStorage 데이터를 JSON 파일로 내보내기/가져오기"
12. **용량 관리** — "localStorage 사용량 표시 + 오래된 데이터 자동 정리"

## 💡 실생활
13. **매일 습관 체크** — "매일 체크하는 습관 리스트, 연속 달성 일수 표시"
14. **출퇴근 기록** — "출근/퇴근 버튼 → 시간 기록 → 일간/주간 통계"
